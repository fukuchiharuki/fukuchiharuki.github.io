方法 †

#!/usr/bin/perl
{
  package Hoge;

  # コンストラクタ
  sub new {
    my $class = shift;
    my $this = {
      foo = 10,
    };
    return bless $this, $class;
  }

  # メソッド
  sub getFoo {
    my $this = shift;
    return  $this->{foo};
  }
}
{
  package main;
  $hoge = new Hoge();
  print $hoge->getFoo() . "\n";
}

解説 †

ポイントは

  • パッケージ分割
  • shift（引数）
  • bless
