Apache/特定のパスへのアクセスをファイルパスにマップする

トップページに戻る

キーワード †

  • Apache
↑

概要 †

特定のパスへのアクセスをファイルパスにマップしたい。

↑

方法 †

例えば/images/〜へのアクセスを/var/images/〜にマップさせるには次のようにします。

Alias /images /var/images
<Directory "/var/images">
Order allow,deny
Allow from all
</Directory>
↑

解説 †

後ろはワイルドカードの扱いです。

アプリケーションが直接wwwな領域を触りに行くのはなんなのでマップして、がいいのかなと思います。

↑
