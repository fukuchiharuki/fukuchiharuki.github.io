概要 †

カンマ区切りのデータを出力します。

↑

方法 †

$ mysql -u(username) -p(password) (database) -B -e 'select * from APPLICATION_EQIX' | \
> sed -e 's/"/\\"/g' | \
> sed -e 's/\t/","/g' | \
> sed -e 's/^/"/' | \
> sed -e 's/$/"/'
↑

解説 †

mysql の各オプションは以下のとおり。

  • -B
    • セパレータとしてタブを使う
  • -e
    • クエリ指定

続けて sed でタブをカンマにしてフィールドを " で囲みます。 わざわざ sed で整形する理由についてはひとつめの参考サイトに書いてあります。
