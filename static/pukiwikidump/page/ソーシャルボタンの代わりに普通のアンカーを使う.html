ソーシャルボタンの代わりに普通のアンカーを使う

トップページに戻る

キーワード †

  • ソーシャルボタン
  • Facebook
  • Twitter
↑

概要 †

自前の画像とかテキストをソーシャルボタン的にしたいときのアンカー(aタグ)。

↑

方法 †

↑

Facebook †

<a href="https://www.facebook.com/sharer/sharer.php?u={PAGE_URL}">Facebook</a>
↑

Twitter †

<a href="https://twitter.com/intent/tweet?text={PAGE_TITLE}&url={PAGE_URL}&via={YOUR_ACCOUNT}">Twitter</a>
↑

解説 †

{PAGE_URL}や{PAGE_TITLE}はJavaScriptで次のように取得できる。

var page_url = document.URL;
var page_title = document.title;

さらに{PAGE_URL}はURIエンコードすること。

encodeURI(page_url)
↑

↑

Twitter †

text

url
シェアさせたいページのURL
via
自分のTwitterアカウント
