シェル/sudoを設定する

トップページに戻る

キーワード †

  • sudo
  • visudo
↑

フォーマット †

root    ALL=(ALL)       ALL

順番に、

  1. 誰が
  2. どこから（通常、ALL）
  3. 誰の権限で
  4. どのコマンドを実行することができるか

を設定する。

↑

設定例 †

  • ユーザーhogeが
  • root権限で
  • なんでもすることができる
hoge    ALL=(root)      ALL
  • グループmogeが
  • root権限で
  • シャットダウンすることができる
%moge   ALL=(root)      /sbin/shutdown
↑

ノート †

  • グループを指定するには「%」を使う
  • 複数のコマンドを指定するには「,」で連結する
↑
