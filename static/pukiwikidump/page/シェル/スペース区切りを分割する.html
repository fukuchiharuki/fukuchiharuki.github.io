シェル/スペース区切りを分割する

トップページに戻る

キーワード †

  • set
  • shift
↑

概要 †

スペース区切りの文字列を分割します。

↑

方法 †

sample.sh:

set -- `echo i1 i2 i3`
while [ $# -ne 0 ]
do
    echo $1
    shift
done

次の実行結果を得ます。

$ ./sample.sh
i1
i2
i3
$
↑

解説 †

$1 や shift は引数を扱うためのものですが set を使うことで引数の代わりにすることができます。
