トップページに戻る

↑

概要 †

次のように実行するときにファイルを一行ずつ処理したい。

./test.sh < file.txt
↑

方法 †

こう書きます。

#!/bin/sh

while read LINE
do
    # ここに $LINE を使った処理
done
↑

解説 †

一行ずつ処理をするなら

awk '{print $1+1}' < /dev/stdin

した方がいいよ、という見方もあるようです。詳しくは参考サイトをご覧になってください。
