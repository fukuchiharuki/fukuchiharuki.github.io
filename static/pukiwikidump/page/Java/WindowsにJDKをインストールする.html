インストールファイルをダウンロードする †

  1. 次のサイトの JavaPlatform(JDK)->JDK を選択する
    • http://java.sun.com/javase/ja/6/download.html
  2. Platformに Windows を選択する
  3. 「I agree to...」にチェックを入れる
  4. Continue を選択する
  5. jdk-6u24-windows-i586.exe（2011-04-27時点）をクリックしてダウンロードする
↑

JDKをインストールする †

  1. ダウンロードしたインストールファイルを実行する
  2. 次へ、完了をクリックしていく
