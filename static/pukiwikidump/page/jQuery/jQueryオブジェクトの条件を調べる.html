概要 †

class属性やid属性からjQueryオブジェクトを引いたとき、そのオブジェクトが何者であるか調べたい。

↑

方法 †

if ($(this).is('input')) {
  $(this).val('ほげ');
} else {
  $(this).html('ほげ');
}
↑

解説 †

jQueryオブジェクトのis()メソッドにセレクタを渡すことでそのセレクタにマッチするかどうかを調べることができます。
