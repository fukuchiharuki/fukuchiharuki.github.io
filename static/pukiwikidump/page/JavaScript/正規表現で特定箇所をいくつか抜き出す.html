概要 †

正規表現で特定箇所をいくつか抜き出したい。

↑

方法 †

var text = "1984-03-12";
var regexp = new RegExp("^([0-9][0-9][0-9][0-9])-([0-9][0-9])-([0-9][0-9])$");
var matcher = text.match(regexp);
if (matcher) {
  year = RegExp.$1; // "1984"
  month = RegExp.$2; // "03"
  day = RegExp.$3; // "12"
}
↑

解説 †

RegExpクラスを使用すると特定箇所を抜き出すことができます。
