概要 †

JavaScriptで、関数の引数が与えられなかったことを判定したい。

↑

方法 †

function func(arg) {
  if (typeof arg == 'undefined') {
    // 引数が与えられなかったときの処理
  }
}
↑

解説 †

関数の引数が与えられないとき引数はundefinedになる。従って、

if (typeof arg == 'undefined')

を書くことができる。

ここで、次の場合ではnullや0と判断がつかなくなるので注意。

if (arg == undefined)
