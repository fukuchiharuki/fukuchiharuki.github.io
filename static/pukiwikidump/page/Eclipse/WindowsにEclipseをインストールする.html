手順 †

↑

アーカイブファイルをダウンロードする †

  1. 次のサイトの Downloads を選択する
    • http://www.eclipse.org/
  2. 「Eclipse IDE for Java EE Developers」右の「Windows 32 Bit」を選択する
  3. 下向き矢印を選択してダウンロードする
    • eclipse-jee-helios-SR2-win32.zip（2011-04-28時点）
↑

エクリプスをインストールする †

  1. ダウンロードしたアーカイブファイルを適当なディレクトリに解凍する
↑

初期化ファイルを編集する †

eclipse.exeを実行すると次のエラーに遭遇することがあります。

  • Failed to create The Java Virtual Machine

このとき、eclipse.ini に次の2行を追記します。※

-vm
C:/Program Files/Java/jdk1.6.0_24/bin/javaw.exe

追記したら再度eclipse.exeを実行してください。

※次の行の上に追記してください。

-vmargs
