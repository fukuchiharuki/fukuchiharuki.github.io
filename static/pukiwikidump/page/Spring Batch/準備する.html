手順 †

↑

STSを手に入れる †

下記サイトからSTSを入手します。

  • http://www.springsource.com/developer/sts
↑

テンプレートプロジェクトを作成する †

下記サイトの Documentation->Getting Started を参考にします。

  • http://static.springsource.org/spring-batch/getting-started.html
  1. File->New->Other... からプロジェクトを作成します
    1. SpringSource Tool Suite->Spring Template Project から
      1. Simple Spring batch Project を選択します
    2. 適当な Project name をつけてプロジェクトを作成します
↑

テンプレートプロジェクトを実行する †

  1. Run->Run Configurations... から実行環境を設定します
    1. 左側 Java Application をダブルクリックして
    2. 右側 Main タブの Main class に次のクラスを設定します 
      org.springframework.batch.core.launch.support.CommandLineJobRunner
    3. 右側 Arguments タブの Program arguments に次を設定します 
      launch-context.xml job1
    4. そのまま右下の Run してみます
  2. Console ビューで次の表示を確認します 
    <Job execution complete: ...
↑
