↑

CPU †

processor と physical id と cpu cores を見てみます。

processor
コアの通し番号、物理×1物理あたりのコア×HT(ハイパースレッディング)が分かる
physical id
物理の通し番号、物理が分かる
cpu cores
1物理あたりのコアが分かる
↑

メモリ †

MemTotal と SwapTotal を見てみます。

MemTotal
メモリ量が分かる
SwapTotal
Swap量が分かる
