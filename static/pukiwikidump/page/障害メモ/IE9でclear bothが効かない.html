障害メモ/IE9でclear bothが効かない

トップページに戻る

キーワード †

  • IE9(Internet Explore 9)
  • HTML/CSS
  • float
  • clear
  • overflow
↑

現象 †

ありがちな「clear: both;」がIE9でなぜか効かない。 正確には片方の下側にしか回らない。

.left {
  float: left;
}

.right {
  float: right;
}

.container:after {
  content: "";
  display: block;
  clear: both;
}
↑

原因 †

分からん。

↑

対策 †

.container {
  overflow: auto;
}

もしかしたら width も入れてあげないとだめかも。 詳しくは参考のサイトを見て。

↑

参考 †
