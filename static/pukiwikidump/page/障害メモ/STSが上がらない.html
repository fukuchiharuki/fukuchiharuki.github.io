障害メモ/STSが上がらない

トップページに戻る

キーワード †

  • STS
  • Eclipse
  • java
  • 64bit
↑

現象 †

ダウンロードしたSTSが上がらない。

Java was started but returned exit code=13

STSもJDKもそれぞれベーシックにしているものをダウンロードした。

  • STS
    • Spring Tool Suite™ (STS) の「Download STS (3.9.1.RELEASE for Windows)」
  • JDK
    • Java SE Development Kit 9 - Downloads の「jdk-9.0.1_windows-x64_bin.exe」
↑

原因 †

64bit用か32bit用かで間違えてる。

↑

対策 †

STSからベーシックにダウンロードできるパッケージは32bit用なので、ダウンロードページから64bit用のパッケージをダウンロードする。

あるいはSTS.ini (eclipse.iniにあたるもの)でどうにかする方法もあるみたいだが、素直に環境に合わせて用意すればいいと思う。

↑

備考 †

JDKはベーシックにダウンロードできるものが64bit用のもの。

↑
