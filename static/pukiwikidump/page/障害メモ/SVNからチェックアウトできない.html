障害メモ/SVNからチェックアウトできない

トップページに戻る

キーワード †

  • Subversion(SVN)
  • チェックアウト
  • エラー
↑

現象 †

SVNからチェックアウトしようとすると次のメッセージでエラーになる。

can't find a temporary directory
↑

原因 †

HDがいっぱいになっているからのようでした。

$ df -h
Filesystem            Size  Used Avail Use% マウント位置
                       50G   49G     0 100% /

100%!!

↑

対策 †

不要ファイルを消します。

$ df -h
Filesystem            Size  Used Avail Use% マウント位置
                       50G   14G   34G  29% /
↑

備考 †

$ df -h

マウントしたディスクごとの容量を確認する。

$ du -k --max-depth=1 /

ファイルシステム上の容量を確認する。深さを1にしてどこが圧迫しているのかを探した。
