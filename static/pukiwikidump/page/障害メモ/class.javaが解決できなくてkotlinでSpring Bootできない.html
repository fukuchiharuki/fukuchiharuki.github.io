障害メモ/class.javaが解決できなくてkotlinでSpring Bootできない

トップページに戻る

キーワード †

  • Kotlin
  • Spring Boot
↑

現象 †

unresolved reference: java

次の箇所で上記のように怒られる。

fun main(args: Array<String>) {
	SpringApplication.run(Application::class.java, *args)
}
↑

原因 †

kotlin-stdlibが足りてない。

↑

対策 †

dependencyに足す。

compile("org.jetbrains.kotlin:kotlin-stdlib:${kotlinVersion}")
↑

備考 †

Springにjavaのクラスを渡すにはclass.javaとする。

↑
