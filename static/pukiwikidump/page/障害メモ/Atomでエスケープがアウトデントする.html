障害メモ/Atomでエスケープがアウトデントする

トップページに戻る

キーワード †

  • Atom
  • ショートカット
  • ctrl-[
  • エスケープ
  • アウトデント
↑

現象 †

「ctrl-[」で入力モードから抜けて、再度「ctrl-[」するとアウトデントしてしまう。連打する癖があるので結構辛い。

インデントの反対はアウトデントなの？

↑

（2016-11-08追記） †

逆に、入力モード中にインデントしてしまう＠Ubuntu。

これについてはこちらの記事を参考にして解決できた。

  • 欅樹雑記: Atomのvim-modeでCtrl+[が効かない！（keyboard-localizationプラグインと日本語キーマップのあれこれ）
↑

原因 †

Atomにキーバインドが設定されているから。

↑

対策 †

  • File > Keymap...

で次を打ち込んでおく。

'atom-workspace atom-text-editor:not([mini])':
  'ctrl-[': 'unset!'
↑

備考 †

  • File > Settings > Keybindings

で設定を見ることができる。

↑

参考 †

  • Atomのキーバインドを無効に(又は変更)する - Qiita
