障害メモ/エンターキーによるフォーム送信抑止が効かない

トップページに戻る

関連 †

  • JavaScript
  • HTML
↑

現象 †

input[type="button"]にしてもエンターキーでフォーム送信する。

↑

原因 †

input[type="text"]がひとつだけの場合そうなる。

↑

対策 †

input[type="text"]をdisplay:none;で隠して置く。

<input type="text" name="dummy" style="display: none;">
↑

参考 †
