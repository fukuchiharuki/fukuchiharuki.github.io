障害メモ/DataTablesの「No data available in table」が邪魔

トップページに戻る

キーワード †

  • DataTables
  • drawCallback
↑

現象 †

データが存在しないと次の内容の行ができてしまう。

No data available in table
↑

原因 †

障害というか、DataTablesの仕様。

↑

対策 †

描画後に無理やり消してやる。

.DataTable({
  .. options ..
  drawCallback: function(settings) {
    $('.dataTables_empty').parent().hide();
  }
});
↑

備考 †

この行を出力しないためのオプションはないっぽい。

↑
