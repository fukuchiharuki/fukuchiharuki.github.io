目的 †

EeePCを使うにあたってはSSD（Solid State Drive）に優しくすると嬉しいらしいです。

↑

tmpディレクトリにメモリ空間をあてる †

/etc/fstabを次のように編集します。

tmpfs   /tmp    tmpfs   defaults,noatime        0       0
